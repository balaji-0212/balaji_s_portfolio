Key Features

  • Overview Dashboard: Comprehensive view of total applications, funded amounts, and received amounts with real-time metrics
  • Loan Status Analysis: Detailed breakdown of Approved, Rejected, and Charged-Off loans with visual representations
  • Key Performance Indicators: Track interest rates, DTI ratios, and loan term distributions across the portfolio
  • Interactive Filters: Dynamic filtering by loan purpose, State/Region, and Grade/Sub-grade for targeted analysis
  • Risk Assessment: Color-coded visual indicators highlighting high-risk loan categories and segments
  • Trend Analysis: Month-over-month and year-over-year comparisons with historical data visualization
