Project Overview

Developed an intelligent energy monitoring system using ESP32 microcontroller that tracks real-time energy consumption and automatically controls appliances based on occupancy detection using PIR sensors.

The system integrates with Ubidots cloud platform for remote monitoring and data visualization, enabling users to track energy usage patterns and optimize electricity consumption from anywhere via mobile dashboard.

Key Features

Real-time Monitoring

Track power consumption with live data updates and instant energy usage feedback

Occupancy Detection

Automatic appliance switching using PIR motion sensors for energy savings

Cloud Integration

Ubidots platform for remote data visualization and mobile dashboard access

Cost Optimization

Reduce electricity bills through automated control and usage analytics
